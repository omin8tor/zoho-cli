# zoho-cli

Command-line interface for Zoho REST APIs (CRM, Projects, WorkDrive, Writer, Cliq).

## Installation

```bash
pip install zoho-cli
```

## Quick Start

```bash
zoho auth login --client-id YOUR_CLIENT_ID --client-secret YOUR_CLIENT_SECRET
zoho crm records list Leads
zoho projects list
zoho drive folders list
```
